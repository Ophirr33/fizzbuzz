DrRacket -> File -> Install Packages -> <Path to fizzbuzz>
or
`raco pkg install <Path to fizzbuzz>`

# Usage
```racket
#lang fizzbuzz

1...100

fizz=3
buzz=5
```

Run with DrRacket or
`racket example.rtk` from commandline
